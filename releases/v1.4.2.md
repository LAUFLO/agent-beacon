# Agent Beacon v1.4.2

## 更新内容

- 修复新版 Codex 终端审批弹窗出现后状态灯仍保持绿色的问题。
- 同时兼容旧版 `custom_tool_call / exec` 与新版 `function_call / exec_command` 会话事件。
- 扩展中文审批提示识别，支持“是否允许……”类命令确认文案。
- 用户确认后会正确从黄灯恢复为绿灯，任务结束后继续按现有规则切换为红灯。
- 新增新版 Codex 真实事件结构回归测试，覆盖等待审批、确认后继续和任务完成状态。

## 升级说明

无需重新配置 TRAE MCP、Claude Code Hooks 或 OpenCode 插件，可直接覆盖升级。
