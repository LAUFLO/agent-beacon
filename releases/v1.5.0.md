# Agent Beacon v1.5.0

## 更新内容

- 新增 Codex 事件格式兼容层，集中兼容新旧终端调用和交互事件名称。
- 补充 Codex Computer Use 权限确认事件识别，等待授权时立即切换黄灯，确认后恢复绿灯。
- 新增确定性构建，同一提交生成的主程序与安装包具有一致 SHA-256。
- 新增集成健康检查与自动修复，可刷新已配置的 TRAE MCP、Claude Code Hooks 和 OpenCode 插件。
- 新增自适应扫描：等待确认时提高响应速度，任务运行时保持实时，空闲时降低后台开销。
- 新增清单化状态规则回归库，覆盖 Codex、Claude Code 和 OpenCode 的脱敏事件样本。
- 新增通知策略：黄灯延迟、同一次确认仅提醒一次、长任务运行提醒。
- 设置页新增当前版本与今日统计，显示完成数量、运行时长和等待时长，不保存聊天正文。
- 设置页集成操作改为紧凑双列布局，并将集成修复和状态中心移入释放出的空间。
- 完善 Per-Monitor DPI 与多显示器边界保护。
- 发布包新增 Portable ZIP 和带开始菜单、卸载入口、自定义安装位置的 Setup EXE。
- GitHub Actions 升级至 Node.js 24 运行时的官方 Action。

## 升级说明

无需重新配置 TRAE MCP、Claude Code Hooks 或 OpenCode 插件，可直接覆盖升级。Setup EXE 支持使用默认当前用户目录或选择自定义安装位置。
