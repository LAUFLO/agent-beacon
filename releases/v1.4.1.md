# Agent Beacon v1.4.1

## 更新内容

- 收紧托盘和任务栏右键菜单宽度，移除右侧多余空白，使绿色选中背景与白色菜单内容区对齐。
- 状态中心由 760×600 缩小为 700×520，并同步优化诊断区、历史区、像素滚动条和底部按钮布局。
- 新增对应界面回归检查，防止菜单宽度和状态中心尺寸退化。

## 升级说明

无需重新配置 TRAE MCP、Claude Code Hooks 或 OpenCode 插件，可直接覆盖升级。

## 安装包

- 文件：`Agent-Beacon-1.4.1.exe`
- 平台：Windows x64
- 运行环境：.NET Framework 4.8
