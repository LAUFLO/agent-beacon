# Agent Beacon v1.6.1

## 更新内容

- 全面更新 Agent Beacon 品牌图标：使用居中的黑色直角灯框与三枚直角状态色块，保持纯白不透明背景。
- 应用主程序、安装包、窗口标题栏、任务栏和系统关联图标统一使用新版多尺寸 ICO。
- README 展示图更新为新版 PNG，并新增同几何、同配色的 SVG 矢量版本。
- Logo 生成脚本改为单一几何源，同时生成 PNG、SVG 和包含 16–256 像素九种尺寸的 ICO，避免不同格式之间出现比例差异。

## 文件

- 单文件：`Agent-Beacon-1.6.1.exe`
- 便携包：`Agent-Beacon-Portable-1.6.1.zip`
- 安装包：`Agent-Beacon-Setup-1.6.1.exe`
